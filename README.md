# <center> Анализ и очистка данных резюме

### Описание проекта

Этот проект посвящён пошаговому разведывательному анализу, очистке и предварительной подготовке выборки резюме для последующего построения моделей и визуализаций рынка труда. Использована база данных резюме, обработаны числовые, категориальные и текстовые признаки, применены методы визуализации, очистки, выявления выбросов и заполнения пропусков.​

### Основные этапы работы

#### 1. Загрузка и преобразование данных
Импорт исходных CSV-файлов с использованием pandas.

* Преобразование дат, денежных и категориальных признаков в удобные для анализа форматы.

* Приведение признака зарплаты к единой валюте (перевод по курсу ЦБ).

#### 2. Первичный разведывательный анализ
   
* Исследование распределения признаков: «Возраст», «Опыт работы (месяц)», «Желаемая заработная плата».

* Построение гистограмм, boxplot, тепловых карт и диаграмм рассеяния.

* Описание моды, медианы, основных интервалов, выявление аномалий.

#### 3. Очистка данных
   
* Поиск и удаление полных дубликатов строк.

* Вывод информации о наличии пропусков в каждом столбце.

* Удаление записей с пропусками в критичных столбцах («Последнее/нынешнее место работы», «Последняя/нынешняя должность»).

* Заполнение пропусков в количественных признаках медианным значением.

* Вручную удалены выбросы по зарплате (выше 1 000 000 или ниже 1 000 руб), по возрасту (анализ в логарифмическом масштабе с помощью z-отклонения), а также резюме, где опыт работы превышает возраст.

#### 4. Анализ ключевых зависимостей
   
* Построение многоуровневых столбчатых диаграмм для анализа связи медианной зарплаты с готовностью к переезду/командировкам.

* Группировка по образованию, возрасту, городу с помощью сводных таблиц и тепловых карт.

* Диаграммы рассеяния для выявления левых/правых аномалий и проверки логических ошибок в данных о возрасте и опыте работы.

#### 5. Выводы и рекомендации
   
* Подробно описаны механизмы аномалий в каждой группе признаков (зарплата, возраст, опыт).

* Категория города и уровень образования — ключевые факторы в модели прогнозирования зарплаты.

* Описан подход к очистке и заполнению на реальных бизнес-запросах.

### Использованные инструменты и библиотеки

* Python 3, pandas, numpy — обработка данных и вычисления.

* Matplotlib, seaborn, plotly — визуализация распределений и зависимостей.

* Jupyter Notebook — оформление кода и аналитической записки.

* Git для управления версиями, README.md для прозрачной структуры проекта.
